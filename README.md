Análise de Veículos - Web App com Streamlit

Aplicação construída em Python para explorar e visualizar dados de veículos de forma simples e interativa.

Descrição

Este projeto é um aplicativo web desenvolvido com Streamlit, criado para visualizar e analisar informações contidas no dataset vehicles.csv.
A aplicação permite filtrar, explorar e entender padrões relacionados a preços, quilometragem e características gerais dos veículos.

O objetivo do projeto é praticar análise de dados e desenvolvimento de aplicações interativas usando Python e Streamlit.


Tecnologias utilizadas

Pandas
Streamlit
SCIPY
enredo
